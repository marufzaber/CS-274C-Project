(1) install a virtualenv (python 3.7)
(2) pip install -r requirements.txt
(3) git clone the fma repo into projects/fma
(4) copy modified_fma_utils.py into projects/fma/utils.py (yes it's hacky!)
(5) download the fma_small dataset and fma_metadata
(6) put AUDIO_DIR and AUDIO_META_DIR in your environment variables
(7) run the project.train.train module (see the cmd line args)
